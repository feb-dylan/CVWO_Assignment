Project Description
This project is a simple discussion forum web application.
Users can create topics, write posts, and comment on posts.

Tech Stack
Frontend: React / Typescript
Backend: Go 
Database: MySQL

Features
- Create Topic
- View Topics
- Create Post
- Comment on Post
- Edit/Delete Posts
- Edit/Delete Topics
- Edit/Delete Comments


How to run
Backend
cd forum-backend
go run main.go

Frontend 
cd forum-frontend
npm install
npm run dev
